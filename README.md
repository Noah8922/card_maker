# Card Maker / 명함관리 앱

거래처 명함 생성 및 관리 앱

## 📆 작업기간

<strong>2022.05.06 ~ 2022.05.24</strong>

## 💡 구현기능

### 소셜 로그인 구현

- firebase 기반의 google, github API

### 명함 만들기

- 사진 업로드 가능 - 로딩 스피너 구현

### 수정 및 삭제 가능

- 내용 수정 및 삭제 후 실시간 데이터베이스와 연동
